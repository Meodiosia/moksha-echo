V1.0 RunManager 验收标准：
✅ 1. "开始新局"→ 状态变 selecting_relic，弹出法宝选择
✅ 2. 选择法宝后 → 状态变 in_room，路线图第1格高亮
✅ 3. "强制清场" → 弹出法宝奖励，选完进入下一房间
✅ 4. 经过4个房间（战斗→宝库→精英→Boss）→ 结算弹出
✅ 5. Boss 房清场 → 触发 run:boss_defeated 事件，日志出现
✅ 6. 宝库房进入即弹奖励（无需清场）
✅ 7. 精英房奖励中必有 rare 法宝
✅ 8. "触发死亡" → 结算弹出 outcome=defeat
✅ 9. 路线图实时反映当前房间和已完成房间
✅ 10. 法宝装备后背包格实时更新
